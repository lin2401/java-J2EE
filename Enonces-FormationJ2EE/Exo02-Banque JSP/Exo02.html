Java JEE - Exercice 2 - JSP

Seconde application Web en Java et manipulation des JSPs sans la JSTL.

Important : cet exercice n'est pas à reproduire dans un projet réel, il est en l'état absolument pas sécurisé.

1 Installation

Importation du projet

Nous allons refaire la même chose que dans l'exercice 01 mais en JSP.

Vous pouvez

  • continuez dans le même exercice, la dépendance dans votre fichier pom.xml ne change pas et reste 
      ...
<dependency>
  <groupId>javax</groupId>
  <artifactId>javaee-web-api</artifactId>
  <version>${version.javaee-web-api}</version>
  <scope>provided</scope>
</dependency>
  ...
  • ou importez celui qui est dans le dossier de l'exercice (en02.jsp).

2 JSP

JSP Client

Renommez et complétez la page src\main\webapp\clients.html afin de la transformer en JSP.

On souhaite afficher la même chose que ce que faisait la ServletClient de l'exercice 1.

Rappel : votre JSP étant un fichier d'IHM elle doit se trouver dans le répertoire src\main\webapp

JSP Compte

Renommez et complétez la page src\main\webapp\comptes.html afin de la transformer en JSP.

Elle doit faire la même chose que ServletCompte.

Test

Si vous avez importez le projet de l'énoncé, n'oubliez pas d'ajouter votre projet au serveur en sélectionnant le serveur puis en cliquant sur le bouton Add

Lancez votre serveur, puis allez sur l'URL qui caractérise votre JSP : http://localhost:8080/NomDeVotreContext/NomDeLaJsp.jsp

Rappels :

  • vous pouvez aussi faire un clic droit sur une de vos pages JSP Run As - Run on server
  • ATTENTION aux majuscules/minuscules
  • le NomDeVotreContext est spécifié dans les propriétés du projet (en principe en02.jsp). Pour retrouver son nom de context root, faire un clic droit sur son projet puis Properties, allez dans Web Project Settings.
  • le NomDeLaJsp.jsp est le nom comple de votre JSP.
